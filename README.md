AI Compliance Checker

This project is an AI-based Compliance Checker designed to analyze data and evaluate compliance using intelligent agents.

The repository contains the fully integrated web application as well as the AI agent scripts used during the compliance checking process.

Project Structure

web_amin/
Contains the complete integrated web application.

all_scripts/
Contains all scripts related to the AI agents.

example_2/outputs/
Contains all outputs generated while running the compliance checking process.

Installation

Both dependency files must be installed:

requirements.txt

requirements1.txt

Running the Application

The web application is built using Flask.
After installing the dependencies, launch the Flask app from the web application folder and access it through your browser.

Notes

All AI logic is located in the all_scripts folder.

Generated results and outputs are stored in example_2/outputs.
